# Дарина | Қыз ұзату

Статичный одностраничный сайт-приглашение для GitHub Pages.

## Публикация на GitHub Pages

1. Создайте репозиторий на GitHub.
2. Загрузите в него файлы из этой папки: `index.html`, `styles.css`, `script.js`, `assets/`, `.nojekyll`.
3. Откройте `Settings` -> `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/root`, затем сохраните.

Через пару минут GitHub покажет ссылку на сайт.
